Macy's: Associate Information Management System (AIMS)

Macy's AIMS system keeps the track of attendance and productivity of its call centers. This system has two functional modules: Agent Dashboard and AIMS Web. Agent Dashboard is a desktop application used by call center agents whereas AIMS Web is web application used by High-level managers. The desktop implementation was based on the repository concept to communicate with WCF services for operation. The web implementation also uses the same repository concept following the Domain Driven Design (DDD) architecture. It was built in MVC framework with the use of MVVM concept where Knockout JS was used for the client side view models.

  • Developed AIMS and Agent Dashboard
  • Did unit testing and provided technical specifications, change logs and other required documents.
